# 🧮 Calculator

## 📌 Project Overview

This project is a **Basic Calculator** developed as **Task 2** of the **CodeAlpha Frontend Development Internship**.

The calculator performs basic arithmetic operations such as addition, subtraction, multiplication, and division. It features a clean user interface with a display screen, interactive buttons, real-time calculations, and keyboard support for an enhanced user experience.

---

## ✨ Features

- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- 🖥️ Interactive display screen
- 🧹 Clear (C) button
- ⚡ Real-time result display
- ⌨️ Keyboard support (Bonus Feature)
- 📱 Fully responsive design

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)

## 🚀 How to Run

1. Clone or download this repository.
2. Open the project folder.
3. Open **index.html** in your preferred web browser.

No installation or additional software is required.

---

## 🎯 Functionalities

- Perform basic arithmetic calculations.
- Display numbers and operators on the calculator screen.
- Clear the display using the **C** button.
- Calculate results instantly using the **=** button.
- Support keyboard input for faster calculations.
