# 🌐 Personal Portfolio Website

## 📌 Project Overview

This project is a **Personal Portfolio Website** developed as **Task 3** of the **CodeAlpha Frontend Development Internship**.

The portfolio showcases my profile, technical skills, projects, resume, and contact information in a clean, responsive, and user-friendly interface. It is designed to provide visitors with an overview of my background, experience, and web development skills. The project follows the Task 3 requirements of creating a responsive portfolio with smooth scrolling, animations, and hover effects. :contentReference[oaicite:0]{index=0}
