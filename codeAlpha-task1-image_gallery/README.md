# 📸 Image Gallery

## 📌 Project Overview

This project is a responsive **Image Gallery** created as **Task 1** of the **CodeAlpha Frontend Development Internship**.

The gallery displays images in a modern grid layout with an interactive lightbox feature. Users can open images in full-screen mode, navigate between them using Previous and Next buttons, and enjoy smooth animations with a responsive design that works across all devices.

---

## ✨ Features

- Responsive image gallery layout
- Interactive lightbox image preview
- Previous and Next navigation
- Hover effects with smooth transitions
- Mobile-friendly responsive design
- Image categories/filters (Bonus Feature)

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)

## 🚀 How to Run

1. Clone or download this repository.
2. Open the project folder.
3. Double-click **index.html** or open it in your browser.

No installation is required.
